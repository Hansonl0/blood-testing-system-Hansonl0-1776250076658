# Blood Testing system

Write a Java program for a Blood Testing system using a class named A1B2 defined in a file called A1B2.java.

Requirements

The program must calculate LDL Cholesterol (LDLC) using the formula:
LDLC = TC - HDLC - TG/5
The program must:
Prompt the user to input the following integer values:
Total Cholesterol (TC)
HDL Cholesterol (HDLC)
Triglyceride (TG)
Perform the LDLC calculation based on the input values.
Display the calculated LDLC value.
The program must interact with the user via the console.
